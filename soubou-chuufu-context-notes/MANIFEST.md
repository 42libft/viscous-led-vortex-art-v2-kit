# soubou-chuufu-context-notes

Codexプロジェクト直下に展開して使うための notes 追加パック。

このZIPは、すでにある `notes/` フォルダへそのまま合流できる構造です。

## 入っているもの

- `notes/context/` — 公式情報・周辺文脈・Liquid Universeシリーズの位置づけ
- `notes/observations/` — 視覚特徴の観察メモ
- `notes/interpretation/` — 作品解釈・タイトル語彙・設計思想
- `notes/implementation/` — shader / architecture への落とし込み仮説
- `notes/references/` — 参照URLと引用マップ
- `notes/experiments/` — 今後の実験メモ置き場

## 使い方

1. プロジェクトルートでZIPを展開する。
2. Codexに `notes/README.md` と `notes/context/00_soubou_chuufu_context_summary.md` を読ませる。
3. 実装判断に迷ったら、`notes/implementation/01_shader_translation_hypotheses.md` と `notes/observations/` を参照させる。

このパックは、公式情報の要約と実装仮説を混ぜたものです。公式見解とユーザー/開発側の解釈を混同しないよう、各ファイル内で区別しています。
