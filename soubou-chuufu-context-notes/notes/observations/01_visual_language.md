# 視覚言語メモ

このファイルは、公式資料とユーザー観察をもとに、二号機が目指すべき視覚特徴を整理したもの。

## 基本構造

作品はシンプルに分解できる。

```text
作品 = 短冊型ディスプレイ + 3つの黒円 + 背景模様 + LED/発光エフェクト + ランダム遷移
```

## 短冊型ディスプレイ

- 黒い空間の中に、縦長の発光体が立っている。
- 全画面に模様を広げるのではなく、発光する範囲が明確に限定される。
- パネル外の黒は、単なる余白ではなく展示空間の暗さとして扱う。
- 作品が「画面」ではなく「光柱」に見えることが大事。

## 3つの黒円

- 単なる黒い穴ではなく、ブラックホール的な重力場に見える。
- 中心は完全に黒い。
- ぎりぎりの淵では、模様が高速に周回軌道しているように見える。
- 周辺では模様が重力レンズのように屈折・圧縮・引き伸ばされる。
- 視覚的には「目」にも見える。
- 3つあることで、観察されている感じ、複数の生物的な眼点、または星系のような重力配置が生まれる。

## 背景模様

背景模様は1種類ではなく、複数の生物的/宇宙的/物質的状態のあいだを変化する。

候補:

- 液体
- 細胞膜
- 油膜
- 夜光虫の群れ
- 宇宙の星雲
- 植物の葉脈
- 昆虫標本的な膜
- 割れた液晶のような界面

## 発光粒

- 蛍、夜光虫、LED、星、ノイズの中間に見えること。
- どれか一つに確定しないことが大切。
- 粒は単なる装飾ではなく、「名もなき光たちの族譜」を表す要素。

## LED感

- LEDドットは人工物性を持つ。
- ただし、人工的すぎると単なるディスプレイ感になる。
- 有機的な模様とLED粒が衝突せず、同じ発光生態系に見えるのが理想。

## 色

重要な色方向:

- 深い青 / 青紫 / 黒
- 夜光虫のような青白い光
- 蛍のような黄緑・白黄
- 細胞膜のようなピンク・緑
- 油膜のような黒・紫・虹色
- 標本的な白・淡色

## 本家っぽさを作る要素

- 暗部が強いこと。
- 色が濁っていること。
- 光が点ではなく、膜や液体に埋まっていること。
- 黒円周辺に重力的な歪みがあること。
- LEDなのに生物っぽいこと。
- 生物なのに人工的な表示装置っぽいこと。
- 画面全体が「分類不能な発光体の標本」に見えること。
