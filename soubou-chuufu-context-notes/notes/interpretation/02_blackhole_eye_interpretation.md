# 黒円 / ブラックホール / 目 の解釈

## ユーザー観察

黒円は、単なる穴や渦ではなく、ブラックホールのビジュアルと挙動に近い。

特に重要な観察:

- 中心は完全な黒に見える。
- 黒円のぎりぎり淵では、模様が超高速に周回軌道しているように見える。
- 重力場の範囲では、周囲の模様が強力な重力によってレンズ状に屈折している。
- 黒点は目にも見える。
- 作品がこちらを見ているような感覚がある。

## 実装上の分解

```text
BlackHole
├── Event Horizon
│   └── 完全に黒い中心。模様を飲み込む領域。
├── Photon Ring / Critical Rim
│   └── 黒円の淵。模様が高速に周回して見える領域。
└── Gravity Field
    └── 周辺模様を屈折・圧縮・引き伸ばす領域。
```

## 重要な設計原則

黒円の「物理」と「視覚効果」は分ける。

- VortexSystem: 位置、速度、壁反射、接近、swap。
- BlackHoleField: lensWarp、orbitWarp、coreMask、ringMask。
- PatternMode: 背景模様生成。
- Composer: 最終合成。

黒円をModeの中に埋め込まない。

## shader仮説

Event Horizon:

- 中心は最終合成で黒く塗る。
- 背景模様より後に適用する。

Photon Ring:

- 中心からの距離が `radius * 0.95` 付近の細い帯に強い接線方向warpを入れる。
- `atan` で角度を取り、時間で高速にずらす。
- 模様そのものを回すのではなく、サンプリング座標を回す。

Gravity Field:

- 中心に近いほど強いが、event horizon内では消える。
- 背景模様のUVに加算する。
- x方向は短冊アスペクトを考慮して距離計算する。

## 目としての黒円

黒円は観測点としても機能する。

実装上は、完全に生物の目に寄せる必要はない。ただし、以下があると「見られている」感が出る。

- 黒中心の絶対的な暗さ
- 周辺リングの湿った光沢
- 3点配置
- ゆっくりした漂い
- 背景模様が黒円に従属して歪むこと

## 避けること

- 黒円をただの黒い円として描く。
- 黒円の周りに単純な白い輪郭だけを置く。
- 背景とは独立した装飾として扱う。
- Modeごとに黒円ロジックを複製する。
