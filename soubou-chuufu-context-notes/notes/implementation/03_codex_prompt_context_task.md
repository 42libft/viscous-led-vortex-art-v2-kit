# Codex用プロンプト: notes読解タスク

最初に実装へ入る前、またはMode追加前に投げるプロンプト。

```text
notes/ 以下を読んで、蒼氓蟲譜の文脈を実装観点で整理してください。

特に以下を確認してください。

1. 公式情報として確かなこと
2. ユーザー観察・解釈として扱うべきこと
3. shader / architecture へ落とし込むべきこと
4. Reference Modeに必要な視覚要素
5. Original Modeに展開してよい要素
6. 実装時に避けるべき誤解

その後、現在のコード構造に対して、どのファイル/モジュールにどの概念を反映するべきか提案してください。

まだ実装には入らず、`notes/experiments/context_reading_report.md` に読解レポートを作成してください。
```

## Mode追加前の短縮版

```text
notes/context, notes/observations, notes/implementation を参照して、次に追加するPatternModeの視覚目標と実装方針をまとめてください。
本家公式情報とユーザー解釈を混同しないでください。
```
