# 蒼氓蟲譜 コンテキスト要約

## 作品の基本情報

正式名は `リキッドユニバース：蒼氓蟲譜 / Liquid Universe: Sōbō Chūfu — An Unclassifiable Entomologia of the Luminous Swarm`。

TOKYO LIGHTS 2026 の Light Art Park / Visible CONNECTION に展示された、落合陽一による新作インスタレーションとして紹介されている。

公式説明では、蛍・夜光虫・LEDという自然の発光体と都市の発光体が、4メートルの光柱の中で重なり合う作品とされる。計算機自然が生成し続ける映像のなかで、自然と人工の境界が曖昧になり、光を新しい視点で捉え直す体験へ導く、という説明がある。

## 公式ステートメントの核

公式ステートメントで重要なのは、次の概念である。

- 蛍、夜光虫、LEDは、どれも「名を呼ばれない光の蒼氓」として扱われる。
- 4メートルの光柱は、自然の発光体と都市の発光体を同じ頁に記す蟲譜である。
- 計算機自然が生成し続ける映像の中で、自然光と人工光の分類は常に失敗する。
- 分類不能であることだけが確かな、名もなき光たちの族譜である。
- 物化するたびに書き換わる。

## 実装資料としての読み替え

この作品を二号機の実装に落とすなら、単なる「綺麗な流体shader」では足りない。

重要なのは以下。

1. 自然光と都市光の分類不能性
   - 蛍のような粒
   - 夜光虫のような青白い発光
   - LEDパネルの人工的な粒状構造
   - それらが同じ発光生態系として見えること

2. 蟲譜 / Entomologia
   - 昆虫標本・生物分類・観察記録のニュアンス
   - しかし分類は失敗し続ける
   - 生物っぽいが、人工物でもある

3. 光柱
   - 作品は平面画像ではなく、縦長の発光体として成立する
   - 黒い展示空間のなかに浮く短冊型/柱型ディスプレイ
   - 周囲の黒は作品の一部として扱う

4. 物化するたびに書き換わる
   - 固定された一枚絵ではなく、常に生成される状態
   - ランダムAuto遷移は、作品の概念に合っている
   - 同じ状態を正確に繰り返すより、生成の一期一会性が重要

## 二号機における最上位方針

二号機は `短冊ディスプレイ + 3つのブラックホール的黒円 + 分類不能な発光模様 + LED粒 + RandomAuto` として設計する。

A: 本家の特定模様を再現するReference Mode群。  
B: 自分たちのOriginal Mode群。  
C: 作品制作ツール化は現時点では対象外。

AとBは対立しない。どちらも同じMode / Preset / Effect構造で扱う。
