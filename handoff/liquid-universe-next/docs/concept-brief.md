# Liquid Universe Concept Brief

## 1. 背景

蒼氓蟲譜の模倣開発はここで終える。対象がリアルタイム生成作品であり、さらに更新され続ける以上、精密に再現しても似て非なるものに留まるため。

次作では、作品の表面だけを追うのではなく、Liquid Universeという概念を自分の解釈で実装する。

## 2. Liquidの解釈

Liquidは、単に水っぽい質感ではない。

ここでは「固定された境界が信頼できなくなる状態」と定義する。

大衆のディスプレイを前提にするなら、作品は短冊形に固定される必要がない。画面の中で枠そのものが変化し、伸び、裂け、滲み、複数に分かれ、また合流してよい。

## 3. 残すもの

黒円は残す。

理由:

- 個人的にブラックホールが好きである。
- 現行プロジェクトで育てた共通VortexSystem / BlackHoleFieldは、次作でも強い核になる。
- 黒円は、境界が信用できなくなる世界の中で、さらに空間を曲げる存在として機能する。

## 4. 変えるもの

模様は落合陽一作品の再現ではなく、自分の美的判断と人工生命アルゴリズムの融合として作る。

方向性:

- 反応拡散。
- 膜。
- 群れ。
- 細胞壁。
- 結晶成長。
- 粘菌的なtrail。
- 発光する微細構造。
- 宇宙と生物の中間に見えるDigital Nature。

## 5. 最初の一文ゴール

黒い展示空間の中で、変化するディスプレイ枠が液体のように呼吸し、その内部では人工生命的な模様が生まれ、黒円が空間を曲げながら、枠と模様と光点が重み付きランダムに別の生態へ移っていくリアルタイム生成作品を作る。
