# 2026-05-27 Phase 6 Reference Observation

対象:

- `photos/IMG_2843.jpeg`
- `photos/IMG_2852.jpg`

観察:

- 黒円: 中心は常に完全な黒。短冊幅に対して芯は控えめで、周囲の渦・リングの方が広い。
- リング: 黒芯の外側に複数層の周回線があり、白く光るというより、周囲の模様が巻き込まれて濃淡の輪になる。
- 模様: 青紫系は葉脈・細胞膜・宇宙雲が混ざり、鉱物系は楕円の殻や標本片が積み重なる。
- 色: 青紫系は深青、シアン、紫、マゼンタが同居する。鉱物系は橙、生成り、淡いシアンの補色が効く。
- LED感: LEDドットは常に見えるが、模様そのものを潰すほど強くはしない。
- 発光粒: 星のような白い粒が散る。粒はランダムで、画面全体に均等すぎない方が自然。

実装に反映したこと:

- `ref-deep-blue-universe` はシアン一色に寄りすぎないよう、深青をベースに暗い筋と白い星粒を足す。
- `ref-pale-mineral` は白一色ではなく、橙の楕円セルと淡いシアンの輪郭を混ぜる。
- 黒円はMode内に入れず、共通 `BlackHoleField` のEvent Horizon / Photon Ring / Gravity Lensを全Modeで使う。
- 参照写真は実行時assetにせず、観察メモだけを実装判断に使う。
