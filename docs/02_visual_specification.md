# 02. 視覚仕様

## 1. 視覚の基本構造

この作品は、単なるフルスクリーン抽象CGではありません。

暗い展示空間に、細長い発光体が立っている。その中で、3つの黒い穴と複数の素材的な模様が干渉している、という構造です。

```text
黒い空間
  └── 短冊LEDパネル
        ├── 3つのブラックホール
        ├── Pattern
        ├── Color
        ├── Move
        ├── 必要な場合だけComposition
        ├── FireflySystem
        └── 必要な場合だけEffect
```

## 2. 短冊パネル

パネルは作品の器です。

要件:

- 画面中央に細長く表示する。
- パネル外は黒でよいが、完全な無意味な余白ではなく、展示空間として扱う。
- パネルの境界は強すぎる枠線にしない。周囲の黒と発光差で見える方がよい。
- panelUvを全Mode共通で使う。

## 3. 黒円 / ブラックホール

黒円は、作品内の最も重要な主体です。

単なる黒い丸ではなく、背景模様の座標系を曲げるブラックホールとして表現します。

### 3.1 Event Horizon

中心の完全な黒。

- ほぼ純黒。
- 模様や発光粒は飲み込まれる。
- 外周は少しだけ柔らかいが、曖昧にしすぎない。

### 3.2 Photon Ring / Critical Rim

黒円のぎりぎり外側の帯。

- 模様が超高速に周回軌道しているように見える。
- 接線方向のwarpを強くかける。
- Modeごとにリングの色や光沢は変わってよい。
- 深青宇宙ではブラックホール感を強める。
- 白地鉱物では標本の穴のように少し抑える。

### 3.3 Gravity Field

黒円周辺の重力レンズ領域。

- 背景模様を屈折させる。
- 模様を圧縮・引き伸ばしする。
- 黒円から離れるほど弱くなる。
- 複数の黒円のfieldは合成される。

## 4. Reference Mode群

Reference Modeは、本家写真から読み取れる見た目を再現するためのMode / Preset群です。

### 4.1 `ref-pale-mineral`

白地、淡いピンク、黄色、青、水彩、鉱物、粘土、膜。

特徴:

- 背景は明るめ。
- 黒円が穴として強く見える。
- 膜の境界は細く、にじむ。
- LED面の粒状感が見えやすい。
- 発光粒は少なめでもよい。

### 4.2 `ref-deep-blue-universe`

深青、宇宙、星、星雲、強い黒円。

特徴:

- 全体は青系。
- 白い星や発光粒が多い。
- 黒円のPhoton Ringが強く見える。
- 重力レンズがもっとも自然に見えるMode。
- LEDドットが星にも見える。

### 4.3 `ref-magenta-green-cell`

マゼンタ、グリーン、細胞膜、サイケデリックな油膜。

特徴:

- 高彩度。
- 細胞膜・水彩の境界が多い。
- 黒い油膜や黒ずみを混ぜる。
- 黒円周辺の歪みが激しくても成立しやすい。

### 4.4 `ref-blue-purple-vein`

青紫、ピンク、葉脈、枝、血管、翅脈。

特徴:

- 縦方向の枝や線がある。
- 背景に葉脈・血管のような分岐が入る。
- 白い発光粒が線の上に乗る。
- 黒円は枝構造を曲げる。

## 5. Original Mode群

Original Modeは、本家観察から抽象化した構造を使って、自分たちの表現として広げるためのModeです。

### 5.1 `orig-black-oil-cell`

黒油膜、細胞、濁り、皮膚、湿った膜。

- 黒と高彩度の対比。
- 黒い流路や水たまりを強く出す。
- 細胞膜の内側に光沢を出す。

### 5.2 `orig-aurora-membrane`

オーロラ、膜、宇宙、液体。

- 青、緑、紫がゆるく流れる。
- 深青宇宙と細胞膜の中間。
- 黒円周辺でオーロラが巻き込まれる。

### 5.3 `orig-insect-specimen`

昆虫標本、翅脈、鉱物、発光点。

- 翅のような線。
- 薄い膜。
- 標本箱的な静けさ。
- 黒円は眼にも穴にも見える。

## 6. Color / Move / Composition

### 6.1 Color

色はPatternに埋め込まず、独立したカテゴリとして扱う。

- メイン色: blue / red / yellow / green / white / purple / pink。
- 配色: sameFamily / sameFamilyBlack / accentColor / contrast。
- Referenceでは写真の主従関係を優先する。
- OriginalではVersion 1で良かった配色を採用してよい。

### 6.2 Move

Moveは背景模様の動き。

- `calm`: 波のような静けさ。
- `normal`: ゆっくりした流れ。
- `active`: 方向性のある流れ、うねり。
- `violent`: 振動、脈動、切替前の激化。

本家に見られる、奥と手前が逆方向に流れるような表現は `counterFlowLayers` として扱う。

### 6.3 Composition

Compositionは任意。必要な見た目だけが使う。

- 全面にPatternが出る。
- 下の方だけ細胞や油膜が出る。
- ワンポイントで黒油膜が溢れる。
- 端から模様が流入/流出する。
- 奥と手前に別レイヤーがある。

ワンポイントはPreset固定ではなく、seed付きRandomで発生位置を決める。

## 7. FireflySystem

白い点々はEffectではなく、作品共通のFireflySystemとして扱う。

- 白〜淡色の発光点。
- 蛍、星、反射、露のように見える。
- ほぼ全Modeに存在する。
- Event Horizonでは消える。
- 密度、明るさ、またたきはPresetやRandomで調整する。

## 8. Optional Effect

EffectはModeをまたいで使う。

Effectは補助質感であり、白い点や黒円リングの本体ではない。

### 8.1 Scanline

- 動画・LED・撮影された画面の質感。
- 高すぎると安っぽくなるので弱め。

### 8.2 Specular

- 濡れた膜、油、ガラス、LED面の反射。
- 細胞膜や黒油膜で重要。

### 8.3 Grain / Pixel Texture

- デジタル画面のざらつき。
- 写真に写ったLED面に寄せる。

### 8.4 Glow

- 全体のにじみ。
- 使いすぎると形が溶けるため控えめ。

### 8.5 LED Grid

- 黒い方眼は撮影時のモアレやカメラ由来の可能性が高い。
- MVPでは作品表現として優先しない。
- 必要になった場合のみ弱く扱う。

## 9. Visual reference運用

見た目で迷ったら、エージェントは次の順番で参照する。

1. `visual-references/honke-original/photos/`
2. `visual-references/honke-original/frames/`
3. `visual-references/self-v1/photos/`
4. `visual-references/self-v1/frames/`
5. `visual-references/*/notes/`

参照時は、画像をそのままコピーするのではなく、次を観察する。

- 画面の明暗比。
- 黒円の大きさと位置。
- 黒円周辺のリング。
- 模様の密度。
- 色の主従。
- 撮影由来のLED/モアレと作品表現の切り分け。
- 発光粒の量。
- 縦方向の流れや枝。
- 動きの方向、激しさ、特殊Move。
- Patternが全面か、部分的か、奥/手前レイヤーか。
- 静止画で見たときの一番強い印象。
